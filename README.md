# KMeansColorSort
Experimental color sorting demo using K-Means for banding.

# Screenshots
![Color Sorting](https://i.imgur.com/T4dfpe7.png)

## Tech Stack
* C# / .NET Core 3.1 / WPF
* Accord.NET (K-Means)
* [Stylet](https://github.com/canton7/Stylet) (MVVM)
* [ModernWpfUI](https://github.com/Kinnara/ModernWpf)
